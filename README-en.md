## Choosing Technology

- JDK：1.8
- Http Components：HttpClient 4.5.6 、Jsoup 1.11.3
- 模拟登陆组件：Selenium
- OCR识别：Tesseract-OCR
- 图形处理：Captcha Recognize
- JSON工具：Jackson 2.9.6
- 日志管理：Log4j 2.11.1
